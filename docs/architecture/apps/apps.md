# 📱 `apps/` フォルダの構成

`apps/` ディレクトリ配下には、ユーザーに直接提供される「フロントエンド（画面・操作側）」のソースコードが入っています。
現在、大きく分けて「PC用Webアプリ」と「iPhone用コントローラーアプリ」の2つが存在します。

## 📍 `apps/` 直下の構造

```text
apps/
 ├── web/                  # PCモニター側で表示するWebアプリ (React + Vite)
 └── ios-controller/       # iPhoneをコントローラーとして使うためのiOSアプリ (Swift)
```

## フォルダ別の詳細

それぞれのアプリケーションの詳細な構造や役割については、以下の専用ドキュメントをご確認ください。

* **[web/ 配下の構造について](./web/web.md)**
  PCのブラウザで開く画面（ホーム、ゲームプレイ、リザルトなど）を構成するソースコードの説明です。

* **[ios-controller/ 配下の構造について](./ios-controller/ios-controller.md)**
  iPhoneで動作するネイティブアプリ（センサー検知、QRコード読み取り、PCとの通信など）の構成説明です。
