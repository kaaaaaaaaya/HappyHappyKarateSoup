# ⚙️ `services/` フォルダの構成

`services/` ディレクトリ配下には、ゲームの裏側でデータのやり取りや処理を行う「バックエンドAPI」のソースコードが入っています。
現在、主にJavaとSpring Bootで作られたサーバーサイドのアプリケーションが存在します。

## 📍 `services/` 直下の構造

```text
services/
 └── backend/              # ゲームのコアロジックやデータ管理を行うJava/Spring Bootサーバー
```

## フォルダ別の詳細

バックエンド側の詳細な構造や役割については、以下の専用ドキュメントをご確認ください。

* **[backend/ 配下の構造について](./backend/backend.md)**
  ユーザーの得点計算、譜面データの管理、クラウド連携などを担当するコードの説明です。
