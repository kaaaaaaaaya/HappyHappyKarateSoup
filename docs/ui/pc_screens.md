- 画面の操作方法が非自明なので余裕があれば優先的にヘルプを設ける

## 5. ゲーム画面（メインプレイ）
* **5-1. ゲーム準備・カウントダウン画面:**
  * **UI要素:** 「スマホをこっち向き（反時計回りに90度）に回して、こうやって持ってね！」というiPhoneの持ち方のイラストを大きく表示する。
  * **進行:** プレイヤーが持ち替えて構える時間を取った後、画面中央で「3, 2, 1, START!」の大きなカウントダウン演出を行う。
* **5-2. 具材をパンチする画面:**
  * **UI要素:** 音ゲー（太鼓の達人など）のように、上から落ちてくる、または左右から流れてくる具材。下部にはグツグツ煮える鍋。
  * **アクション:** ヒット判定ラインに具材が重なった瞬間に、iPhoneから「パンチ信号」を受信すると具材が真っ二つに割れて鍋に入る。
* **5-3. [優先順位低] 具材をかき混ぜる画面:**
  * パンチフェーズ終了後、iPhoneをぐるぐる回す動作でスープを完成させるフェーズ。

## 5. リザルト画面
* **UI要素:**
  * **右上の配置:** ランク（S, A, B, C）と、スコアの具体的な数値。
  * **中央の配置:** 完成したハッピースープのイラスト。
  * **中央右の配置:** 6角形のレーダーチャート（甘味, 塩味, 苦味, 酸味, うまみ, 辛味）。
  * **画面下部の配置:** AI（Gemini等）が生成した、そのスープに対する「一言コメント＆ハッピーポエム」。
* **アクション:** iPhoneからのコントローラー操作を受け付け、「もう一度遊ぶ」「ホームに戻る」などのメニューを操作できるようにする。
