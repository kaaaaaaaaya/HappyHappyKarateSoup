# 🎨 スタイルガイド (Style Guide)

このドキュメントは、「Happy Happy Karate Soup」プロジェクトの視覚的スタイルとデザイン・ルールを定義します。

## 1. 全体テーマ (Overall Theme)
* **テーマ名:** Neo-Pop Pixel (淡いネオ・ポップ × モダン・ピクセルアート)
* **コンセプト:** 「Wiiのような体感レトロゲーム」を、現代的で淡くポップな色彩で表現。クリーンな白系の背景に、パステル調のドット絵（ピクセルアート）と太い黒枠を組み合わせることで、懐かしくも新しい「エモくて可愛い」UIを目指します。

## 2. カラーパレット (Color Palette)
全体的に彩度を少し落とした「淡い（Pale）」カラーを使用し、ピクセルアートのキツさを和らげます。

### 2.1 共通・背景 (Common & Background)
* **背景色:** `#FFFDF9` (オフホワイト) - アプリ全体のメイン背景。目に優しい色。
* **枠線・文字:** `#1A1A1A` (ほぼ黒) - ピクセルアートの縁取り、UIの太枠、高コントラストなテキスト。

### 2.2 ポップ・カラー (UI & ドット絵のベース色)
* **サーモンオレンジ:** `#FF8C66` - タイトルロゴ、決定ボタン、鍋の火などの強調色。
* **ミントグリーン:** `#6ED6BA` - サブボタン、成功時のエフェクト、リザルト画面のアクセント。
* **ライトイエロー:** `#FFD18C` - カーソル（手）、背景の装飾図形。
* **ペイルブルー:** `#98E8FC` - 選択状態の背景、テキストボックスの内側。

### 2.3 チャート・フィードバック専用
* **六角形レーダーチャート:** * 甘味 (ピンク): `#FFB3BA`
  * 塩味 (水色): `#BAE1FF`
  * 苦味 (緑): `#BAFFC9`
  * 酸味 (黄): `#FFFFBA`
  * うまみ (オレンジ): `#FFDFBA`
  * 辛味 (赤): `#FF9B9B`
* **スコア・コンボ:** `#FF5722` (鮮やかなオレンジ) 

## 3. タイポグラフィ (Typography)
レトロゲーム感を出すため、**ピクセルフォント（ドット文字）**を全面的に採用します。

* **メインフォント (日本語):** `DotGothic16` (Google Fontsで利用可能)
  * 用途: AIのハッピーポエム、メニュー名、各種説明文。
* **メインフォント (英数字・スコア):** `Press Start 2P` または `VT323` (Google Fonts)
  * 用途: 「3, 2, 1, START!」のカウントダウン、SCORE、RANK、"PUNCH!" などの巨大な文字。

## 4. UIコンポーネントのルール (UI Components)
* **枠線と影:** ボタンやパネルには、**2〜4pxの太い黒枠**を付けます。影はぼかさず、右下にズレた**ベタ塗りの黒い影**（`box-shadow: 4px 4px 0px #1A1A1A`）をつけて、シールのような立体感を出します。
* **角丸の禁止（またはピクセル風角丸）:** ピクセルアートの世界観を守るため、CSSの滑らかな `border-radius` は極力使わず、直角（0px）にするか、ドット絵で描いたようなカクカクの角丸を使用します。
* **カーソル:** PC側をiPhoneで操作する際のカーソルは、Wiiをオマージュした「ピクセルアートの白い手（指差し/グー）」のアイコンを使用します。

## 5. アニメーション & フィードバック (Motion & Feedback)
* **視覚エフェクト (PC側):** * 具材を斬った瞬間、ピクセル（四角いドット）が弾け飛ぶパーティクルエフェクトを実装。
  * 「PUNCH!」などの文字は、表示時に画面が少し揺れる（スクリーンシェイク）演出。
* **触覚・聴覚 (iPhone側):**
  * PC側で具材がヒット（切断）した瞬間、WebSocket経由で信号を受け取り、iPhoneの **Taptic Engine（`UIImpactFeedbackGenerator(style: .heavy)` など）** を最高強度で鳴らす。
  * 同時に「スパーン！」またはレトロな「ピシュゥゥン！」という8bit風の効果音をiPhoneから鳴らす。
