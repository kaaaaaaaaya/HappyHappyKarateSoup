# UI Style Guide

本ドキュメントでは、Figmaより抽出したデザインシステム・スタイルの指針をまとめます。
（`pc_screens.md` に記載された各画面IDのFigmaデータに基づいています）

## 1. タイポグラフィ (フォント)
ゲームの雰囲気を出すため、ピクセルアート風（ドット絵風）やレトロ画面を意識したフォントが複数使われています。

* **Press Start 2P** (Weight: 400)
  * 見出しや重要なボタンに使用。サイズ: 18px, 20px など
* **DotGothic16** (Weight: 400)
  * 日本語テキスト、説明文、ボタンラベルなどで主に使用。サイズ: 14px, 18px, 20px, 36px など
* **VT323** (Weight: 400, 500)
  * 英数字のサブテキストや補助的なラベルに使用。サイズ: 18px, 20px など
* **Space Grotesk** (Weight: 500)
  * モダンなサンセリフ体。QRコード周辺の小さな指示文などに使用。サイズ: 14px

## 2. カラーパレット

### メインカラー・アクセント
* **Red/Orange**: `#c36453` / `#ff8686` (ボタンの背景色やアクセントカラー)
* **Teal/Green**: `#74baa4` (タイトル画面などでの装飾)
* **Yellow/Gold**: `#e3d089` (タイトル画面などでの装飾)

### ベースカラー・テキスト
* **White**: `#ffffff` (背景、テキスト)
* **Black**: `#000000` (テキスト、シャドウ)
* **Dark Brown**: `#1d0c0d` (濃い背景色/テキスト色)

### グレースケール (UIパーツ・ボーダー・補助テキストなど)
* **Slate 900**: `#0f172a` (主要テキスト色など)
* **Slate 600**: `#475569` (セカンダリテキスト)
* **Slate 500**: `#64748b` (プレースホルダーや非活性テキスト)
* **Slate 400**: `#94a3b8` (ボーダーなど)
* **Slate 100**: `#f1f5f9` (カード背景や入力欄背景など)
* **Grayish Brown**: `#c6bbb9` (一部の枠線や非活性色)

## 3. レイアウト・形状

### 角丸 (Border Radius)
コンポーネントの役割によって以下の角丸（Corner Radius）が使い分けられています。
* **8px**: 小さなカードや入力要素など
* **16px**: 中くらいのコンテナ、QRコード背景など
* **24px**: ダイアログやメインのポップアップカード
* **9999px (Pill)**: 「ログインせずに遊ぶ」などの角丸ボタン、バッジなど
