# 🎨 スタイルガイド (Style Guide)

このドキュメントは、「Happy Happy Karate Soup」プロジェクトの視覚的スタイルとデザイン・ルールを定義します。
全体を通して、**清潔感（Pale）**と**元気さ（Pop）**、そして太い縁取りによる**レトロ・クール（Neobrutal）**のバランスを目指します。

## 1. 全体テーマ (Overall Theme)
* **テーマ名:** Neo-Pop-Brutalist-Light (淡いネオ・ポップ・ブリュタリスト)
* **コンセプト:** クリーンな白系の背景に、淡くも主張するポップなカラーを配し、太い黒枠で全体を引き締める。レトロゲームのような元気さと、現代的な可愛らしさを両立させる。

## 2. カラーパレット (Color Palette)
※ カラーコードは実装時に微調整する可能性があります。

### 2.1 共通・背景 (Common & Background)
* **背景色 (Background):** `#FFFDF9` (オフホワイト) - アプリ全体のメイン背景。
* **枠線・文字 (Borders & Text):** `#1A1A1A` (ほぼ黒) - すべての外枠、高コントラストなテキスト。

### 2.2 ポップ・カラー (Pop Colors)
ベースとなる淡いカラー群です。図形やボタンの背景に使用します。
* **オレンジ (Orange):** `#FF8C66` - タイトル、メインの決定ボタン、強調したい領域。
* **ミントグリーン (Mint):** `#6ED6BA` - 「PUNCH!」画面、サブの背景図形、"DELICIOUS!"テキスト。
* **イエロー (Yellow):** `#FFD18C` - 背景図形、サブボタン、アクセント。
* **ペイルブルー (Pale Blue):** `#98E8FC` - 淡いテキスト要素、背景図形、リザルト画面の枠。

### 2.3 スコア・フィードバック (Score & Feedback)
* **巨大スコア数値:** `#FF5722` (鮮やかなオレンジ) 
* **ラベル・補足テキスト:** `#666666` (ダークグレー)
* **スコア倍率:** `#FF3B30` (赤) - x1.5などの倍率表示。

## 3. タイポグラフィ (Typography)
* **タイトル・見出し:** `Poppins` または `Montserrat` (太字 / Bold)。
  * 用途: アプリタイトル、"DELICIOUS!"などの大きな英字。大文字（All-caps）推奨。
* **本文・ラベル:** ゴシック体（`Noto Sans JP` など）＋ 英数字は `Space Mono` などの等幅フォント。
  * 用途: 説明文、フォームのラベル、小さなテキスト詳細。
* **スコア (Score):** 太く視認性の高いサンセリフ体（`Roboto Bold` や `Impact` など）を使い、巨大な数字を強調。

## 4. UIコンポーネントのルール (UI Components)
画像の「カートゥーンっぽさ」を出すための最重要ルールです。

* **枠線 (Borders):** すべての主要なカード、ボタン、パネルには、常に **2px〜3px厚の太い黒枠** を付ける。
* **角丸 (Corner Radius):** カード、ボタン、パネルには、**16px** 程度の大きめな角丸
