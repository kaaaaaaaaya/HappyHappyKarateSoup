# 📄 HappyHappyKarateSoup ドキュメント

この `docs` ディレクトリには、PC（モニター役）とiPhone（コントローラー役）を連動させて遊ぶ体感型アクションゲーム「HappyHappyKarateSoup」の仕様書やUIデザイン案をまとめています。

ハッカソンでの迅速な開発と、チーム内の認識合わせを目的としています。

## 📁 ディレクトリ構成とファイル説明

現在のドキュメント構成は以下の通りです。

```text
docs/
 ├── README.md               # 📍 現在のファイル（ドキュメント全体の目次と説明）
 │
 └── ui/                     # 🎨 UI・画面仕様に関するドキュメント
      ├── overview.md        # ゲーム全体のテーマ、世界観、PCとiPhoneをまたいだ画面遷移図
      ├── pc_screens.md      # PC側（モニター）の各画面の詳細仕様と必要なアニメーション
      └── iphone_screens.md  # iPhone側（コントローラー）の各画面の詳細仕様と触覚フィードバック（振動・音）
```
